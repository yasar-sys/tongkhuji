# টঙ বিডি – বাংলাদেশের টঙ মানচিত্র (TongBD)

বাংলাদেশের সেরা চায়ের দোকানগুলো খুঁজে বের করুন। Discover the best tea stalls across Bangladesh.

## Project Description

TongBD is a platform to explore and locate tea stalls (Tongs) across Bangladesh.

## Tech Stack

- **Frontend**: React (with Vite)
- **Styling**: Tailwind CSS, shadcn/ui
- **Maps**: Leaflet (react-leaflet)
- **Backend**: Supabase
- **Language**: TypeScript

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```
